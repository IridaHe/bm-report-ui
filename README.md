# BMUI

Bluemoon Moon UI (currently based on Vue)

## Demo

[here](https://viyayaya.github.io/bmui)

## Development

```
  npm run dev
```

## Production

```
  npm run build
```

# Usage

```
  npm install bmui
```

``` javascript
  import 'bmui' // CSS
  import Bmui from 'bmui/dist/vue' // Vue components
  import Vue from 'vue'
  Vue.use(Bmui)
```

# Example

Please refer to [the demo for vue](https://viyayaya.github.io/bmui)
